Backend scaffold for auth, invitations, and region-based posts.

Quick start

1. Copy `.env.example` to `.env` and fill values.
2. Install:

```bash
npm install
```

3. Run DB migrations (use your preferred tool) using `migrations/init.sql`.
4. Seed super admin:

```bash
npm run seed
```

5. Start server:

```bash
npm run dev
```
